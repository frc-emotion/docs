# Introduction

## What is Nautilus?

A **simple**, **powerful** and **flexible** app for keeping track of your members' attendance, scouting at competitions, and streamlining team communication. It consists of a cross-platform mobile app powered by Kotlin Multiplatform and a backend REST API built on Express.js and MongoDB.

## Get Started

Read through the docs on this website, then when you're ready [clone the project](https://github.com/nautilus-frc/nautilus) and deploy it to your own server using our CLI.
